# Extreme Electronics Website

This is the official commercial engineering portfolio and business website for **Extreme Electronics**. It serves as a digital storefront to showcase the agency's end-to-end engineering capabilities, including Artificial Intelligence / Machine Learning, Electronics, Mechatronics, and Custom Software Solutions.

## Features

- **Modern & Responsive Design**: Built natively using Tailwind CSS to ensure a seamless experience on both mobile and desktop screens.
- **Industrial Aesthetic**: A professional, dark-themed styling tailored to represent high-tech engineering and manufacturing standards.
- **Solution Showcases**: Clean, dynamic layout grids displaying both successfully delivered products and concept-ready innovations.
- **Brochure-Aligned Messaging**: All USPs, contact details, and service descriptions strictly mirror official business brochures.

## Tech Stack

- **HTML5** (Structure and semantic layout)
- **Tailwind CSS via CDN** (Utility-first styling and responsiveness)
- **Vanilla JavaScript** (Mobile navigation and accordion interactions)
- **Google Fonts** (Inter font family)
- **Heroicons** (Scalable vector icons)

## Deployment

This website is fully static, making it extremely lightweight and fast. It is configured for free, automated deployment via **GitHub Pages**.

To publish updates:
1. Make your edits locally to `index.html`, CSS, or JS files.
2. Commit and push the changes to the `main` branch.
3. GitHub Pages will automatically build and deploy the latest version to the live domain.

## Local Development

No complex build steps or node modules are required to view the site locally.

1. Clone or download this repository.
2. Open `index.html` directly in any modern web browser.
3. (Optional) For a better development experience with hot-reloading, use a local server like the [Live Server extension for VS Code](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).
